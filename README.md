# Excel-Sales-Performance-Dashboard-An interactive Excel dashboard analyzing sales trends, profit margins, and KPI performance (2019-2022).
Sales Performance Analysis Dashboard (2019–2022)
📊 Project Overview
This project features a comprehensive Interactive Sales Dashboard built in Microsoft Excel. The primary objective was to transform raw sales data from 2019 to 2022 into a visual story that helps stakeholders monitor performance, track profitability, and identify top-performing products and regions.

🚀 Key Features
Dynamic Interactivity: Integrated multi-dimensional slicers (Year, Country, Item Name) for real-time data filtering.
Trend Analysis: Comparison of Sales Trends and Gross Rates over a 4-year period.
Profitability Tracking: Visual representation of Profit Margins across categories like Laptops, Mobiles, and Home Appliances.
KPI Monitoring: A comparative bar chart showing Net Amount vs. Target for various customer accounts to evaluate performance against goals.
Channel Insights: Distribution analysis of sales across E-Commerce, Retail, Key Accounts, and Wholesale.

🛠️ Tools & Techniques
Data Cleaning: Used Power Query to clean and transform the dataset.
Data Modeling: Established relationships between different data tables.
Pivot Tables & Charts: For dynamic data aggregation and visualization.
Advanced Formulas: Calculated Fields for Margins and Growth Rates.

📁 Project Structure
Sales_Dashboard.xlsx: The main Excel file containing the dataset, pivot tables, and the visual dashboard.
Screenshots/: A folder containing images of the dashboard for quick preview.

💡 Insights Gained
Identification of the highest-performing year and the factors contributing to the gross rate fluctuations.
Deep dive into product categories to identify which items drive the most profit vs. volume.
Geographical analysis to understand market penetration across different countries.
